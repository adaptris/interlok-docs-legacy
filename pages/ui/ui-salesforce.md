## Prerequisite ##

To be able to generate salesforce xml templates you need to have the required adp-apache-http [optional component installed](adapter-optional-components.html).

## Getting Started ##

To access the Salesforce config page, you should use the Salesforce button on the header navigation bar. The page is only accessible by admin and user roles.
