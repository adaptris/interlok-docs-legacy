## Version 3.6.0 ## Release Date : 2017-03-03 This release updates jetty from 8.1.x to 9.4.x; as a result you will need java 8 to run the adapter. The jetty configuration has also changed quite a bit; if you have customised your jetty.xml then you will need to cross reference it with the jetty.xml that is shipped with Interlok. Java 7 has been EOL'd and we are expecting Java 9 to be released in the next few months; we anticipate that most businesses will already be looking to upgrade/are already running Java 8. In the event that there are enough customer requests for it, then we will downgrade to jetty 9.2.x which still works with java 7. The changes for [INTERLOK-1395](https://adaptris.atlassian.net/browse/INTERLOK-1395), [INTERLOK-512](https://adaptris.atlassian.net/browse/INTERLOK-512) and [INTERLOK-1394](https://adaptris.atlassian.net/browse/INTERLOK-1394) may cause your configuration to be invalid so please take care when upgrading. ### Bugs -
