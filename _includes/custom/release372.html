## Version 3.7.2 ## Release Date : 2018-05-04 ### Key Highlights - Service-Tester UI, You are now able to build unit tests for your Interlok Config from within the UI and you can export the configured unit tests for use in your CI system. - The UI config settings editor now has a new sidebar that can be used to view metadata on any given connectable database. - Support added for "mime-encoded" failed messages that are large - New optional component for MongoDB for: Create, Update and Delete operation -
